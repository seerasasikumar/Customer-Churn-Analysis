# Dashboard

## Overview

This folder contains dashboard screenshots for the Customer Churn Analysis project.

## Dashboard Features

- Total Customers KPI
- Churn Customers KPI
- Churn Rate KPI
- Churn by Contract
- Churn by Payment Method
- Monthly Charges vs Churn
- Churn Distribution
- Interactive Slicers

## Filters

- Gender
- Contract
- Payment Method

Customer_Churn_Dashboard.png

## Business Insights

- Month-to-month contract customers have the highest churn.
- Electronic Check is the most common payment method among churned customers.
- Customers with higher monthly charges are more likely to churn.
- The overall churn rate is approximately 26.5%.
- Long-term contracts show better customer retention.
