# Python Analysis

## Overview

This folder contains the Python scripts used for analyzing the Customer Churn dataset. Python was used for data cleaning, exploratory data analysis (EDA), and data visualization to uncover customer churn patterns.

## Objective

The objective of this analysis is to identify factors influencing customer churn and generate meaningful insights through data analysis and visualization.

## Dataset

- Customer Churn Dataset
- Total Records: 7,043

## Libraries Used

- Pandas
- NumPy
- Matplotlib

## Analysis Performed

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Churn Analysis
- Contract Analysis
- Payment Method Analysis
- Monthly Charges Analysis
- Data Visualization

## Visualizations

- Churn Distribution
- Contract-wise Churn
- Payment Method Analysis
- Monthly Charges Distribution

## Key Insights

- Month-to-Month contract customers show the highest churn.
- Customers paying through Electronic Check have a higher churn rate.
- Higher monthly charges are associated with increased customer churn.
- Long-term contracts improve customer retention.

## Output

The Python analysis provides valuable insights into customer behavior and supports business decisions aimed at reducing customer churn.
