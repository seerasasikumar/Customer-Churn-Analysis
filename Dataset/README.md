# Dataset

## Overview

This folder contains the Customer Churn dataset used for the Customer Churn Analysis project.

## Dataset Name

Customer Churn Dataset

## Description

The dataset contains customer information collected from a telecom company. It includes customer demographics, contract details, payment methods, monthly charges, total charges, tenure, and churn status.

## Dataset Information

- Total Records: 7,043
- Dataset Format: CSV
- File Name: Customer_Churn.csv

## Features

- Customer ID
- Gender
- Tenure
- Contract
- Payment Method
- Monthly Charges
- Total Charges
- Churn

## Purpose

The dataset is used to analyze customer churn behavior, identify the factors affecting customer retention, and generate business insights using Excel, SQL, Python, and Power BI.

## Tools Used

- Microsoft Excel
- MySQL
- Python
- Power BI

## Project Objective

To identify churn patterns, understand customer behavior, and support data-driven business decisions for improving customer retention.
