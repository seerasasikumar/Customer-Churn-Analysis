# Customer Churn Dataset

## Overview

This dataset contains customer information collected from a telecom company. It is used to analyze customer churn patterns and identify the factors influencing customer retention.

## Dataset Information

- Total Records: 7043
- Total Columns: 8

## Features

- Customer ID
- Gender
- Tenure
- Contract
- Payment Method
- Monthly Charges
- Total Charges
- Churn

## Project Goal

The goal of this project is to understand why customers leave the company and provide business recommendations to improve customer retention.
